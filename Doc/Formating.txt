Grobe Erläuterung
Jeder GameServer hat genau eine Lobby. Sollte die Lobby Privat sein, so wird ein passwort beim Verbinden verlangt. Matchmaking sucht automatisch eine Freien Server und stellt dort die Lobby entsprechend anforderungen ein (mal schauen ob hier ein gruppen feature geht). Später sollen auch invites möglich sein (Startet einfach verbindung mit Server und bekanntem feature)
Settings:

ServerData;


PlayerData: {GlobalId, PrefColor, Name, StubId, { {Name,StubId}, {GlobalId} } } (Permanent)
Rating:     {UlId (LevelData) ,GlobalId (PlayerData),Rating ,{UlId, GlobalId} }
Server:   {LobbyId, IP, Region, Availability, {LobbyId} }
Invitations: {InvitationId, LobbyId (LobbyData) , GlobalId (PlayerData), {InvitationId, LobbyId} }
LevelData: {Name, UploadDate, Template, UlId, GlobalId (PlayerData) , {UlId} } (Permanent)
